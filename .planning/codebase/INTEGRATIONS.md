# Integrations

- **GPS/Location**: Used for real-time speed monitoring and map positioning.
- **Camera Feed**: Front camera utilized at low resolution for eye tracking.
- **ML Kit Face Detection**: Planned integration for eye openness analysis.
- **Text-to-Speech (TTS)**: Used for vocal warnings (e.g., speed limit exceeded).
- **OpenStreetMap (OSM)**: Map tiles provided via OpenStreetMap.
