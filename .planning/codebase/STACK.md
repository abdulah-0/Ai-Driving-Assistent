# Technology Stack

- **Framework**: Flutter (Dart SDK >=3.0.0)
- **State Management**: None (StatefulWidgets currently), but planned to use Provider.
- **Location Services**: geolocator, flutter_map, latlong2.
- **Hardware Integration**: camera (camera package).
- **AI/ML**: google_mlkit_face_detection (added to dependencies, implementation pending).
- **Audio/Voice**: flutter_tts (TTS), audioplayers (planned for alarms).
- **Permissions**: permission_handler.
